jquery-print-preview-plugin

jQuery plugin for print preview (modal window)

The jQuery Print Preview plugin is designed to provide visitors with a preview of the print version of a web site. Unlike a popup window this plugin brings in all content and print styles within a modal window.

Dependencies

jQuery + a print stylesheet

Usage

  1. Include the CSS and iamges from src/css into your screen stylesheet.
  2. Include src/jquery.print-preview.js
  3. Add a print preview link and initalise the plugin like so 
    $('#foo').prepend('Print this page');
$('a.print-preview').printPreview();
